# Keyboard Shortcuts

## Data Editing (Preview Mode)

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` (Windows/Linux)<br>`Cmd+Z` (Mac) | **Undo** | Undo the last cell edit |
| `Ctrl+Shift+Z` (Windows/Linux)<br>`Cmd+Shift+Z` (Mac) | **Redo** | Redo the previously undone edit |
| `Ctrl+Y` (Windows/Linux)<br>`Cmd+Y` (Mac) | **Redo** | Alternative redo shortcut |

## Tips

- **Undo/Redo History**: The application maintains up to 50 states in history
- **Keyboard Hints**: Look for keyboard shortcut badges in the UI
- **Focus Management**: Click on any cell to start editing, press Tab to move to the next cell
- **Scroll Performance**: Use mouse wheel or trackpad for smooth scrolling through large datasets
